# Einkaufsliste

## Beschreibung
Diese App zeigt Läden in einer Spalte, Waren des jeweiligen Ladens in einer zweiten Spalte.

## Funktion
Abbildungen werden hier eingesetzt.

## Anwendung
Einige Eigenschaften lassen sich zu anderen Zwecken verwenden.
